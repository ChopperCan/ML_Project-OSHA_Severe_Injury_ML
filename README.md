# OSHA Severe Injury Prediction using Machine Learning

This repository contains the complete machine learning course project developed for
predicting permanent injury outcomes using OSHA severe injury reports.

## Project Overview

The main objective of this project is to predict **permanent injury outcomes**
based on OSHA severe injury records using supervised machine learning methods.

Target variable:
- Permanent_Injury (binary classification)
